# CAITG AI Winter School - Software setup

This repository holds the software setup instructions for the CAITG AI Winter School (27-30 July 2026, Camperdown campus).

## Start here: [Software setup guide](software-setup.md)

The guide lists every tool we will install and set up during the school, with links and step by step instructions for each.

There is no need to install anything in advance, as we do it together in class. Please read the short safety note at the top of the guide before the event, and if you cannot install software on your own computer (for example a locked-down or institution-managed laptop), arrange administrator rights beforehand or bring a machine you can install on.

If you get stuck during setup, you can also ask the Winter School installation assistant, linked inside the guide.
